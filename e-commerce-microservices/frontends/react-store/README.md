# React Store

Customer-facing e-commerce store built with React.js, Vite, and Tailwind CSS.

## Features

- Modern React with Hooks
- Responsive design with Tailwind CSS
- Product browsing and search
- Shopping cart functionality
- User authentication
- Real-time chat integration with Socket.io
- Fast development with Vite

## Pages

- **Home** - Landing page with key features
- **Products** - Product listing with filters
- **Product Detail** - Individual product view
- **Cart** - Shopping cart management
- **Login/Register** - User authentication

## Components

- **Navbar** - Navigation header
- **Chat** - Real-time customer support chat widget

## Tech Stack

- React 18
- React Router v6
- Axios for API calls
- Socket.io-client for WebSocket
- Tailwind CSS for styling
- Heroicons for icons
- Vite for build tooling

## Running Locally

```bash
npm install
npm run dev
```

Access at: http://localhost:3000

## Building for Production

```bash
npm run build
npm run preview
```

## API Integration

Connects to backend services through API gateway at http://localhost:80
